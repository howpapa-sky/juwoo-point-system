# 유사 앱 조사 결과 및 기능 분석

## 조사한 주요 앱들

### 1. Rewards Kids
**주요 기능:**
- 포인트 시스템: 좋은 행동에 대한 포인트 적립
- 가상 저금통 (Virtual Piggy Bank): 용돈 관리
- 스크린 타임 관리: 포인트로 스크린 타임 연장 가능
- 부모-자녀 앱 분리: 부모는 승인, 자녀는 요청
- 자동 용돈 스케줄링
- 실시간 알림 시스템
- 타임라인 기능 (포인트/돈 내역)

### 2. Goody Points
**주요 기능:**
- 타이머 기능: 시간 도전 과제
- 간단한 포인트 추가/차감
- 목표 설정 및 달성 축하
- 연장 시간 (Overtime) 기능
- 실시간 클라우드 동기화
- 시각적 카운트다운

### 3. Child Reward / NeatKid
**주요 기능:**
- 집안일 추적기
- 과제별 포인트 설정
- 보상 차트
- 사진 업로드 (완료 증명)
- 진행 상황 추적
- 맞춤형 보상 설정

### 4. 아이쿠카 (한국 앱)
**주요 기능:**
- 쿠카포인트 보상 시스템
- 해냄/잔돈 통장
- 목표 설정 및 저축
- 금융 교육 요소

## 영어 학습 앱 기능 분석

### Duolingo / Lingokids
**주요 기능:**
- 게임화된 학습
- 일일 목표 및 연속 학습 (Streak)
- 레벨 시스템
- 배지/업적 시스템
- 인터랙티브 플래시카드
- 음성 인식 및 발음 연습
- 진도 추적

### Studycat English
**주요 기능:**
- 나이별 맞춤 콘텐츠 (2-8세)
- 인터랙티브 레슨
- 게임 기반 학습
- 시각적 학습 자료

## 우선순위

### Phase 1 (즉시 구현)
1. 영어 단어 학습 기능 (플래시카드, 퀴즈)
2. 포인트 타임라인
3. 기본 통계 대시보드

### Phase 2 (다음 단계)
1. 목표 설정 기능
2. 배지/업적 시스템
3. 타이머 기능

### Phase 3 (향후)
1. 알림 시스템
2. 부모 대시보드
3. 고급 통계 및 리포트
