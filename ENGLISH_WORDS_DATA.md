## 영어 학습 시스템 설계

### 학습 모드

**1. 플래시카드 모드**
- 영어 단어 표시 → 클릭 → 한글 뜻 표시
- 다음/이전 버튼으로 이동
- 진행률 표시 (예: 3/10)

**2. 퀴즈 모드**
- 영어 단어 표시
- 4개의 한글 뜻 선택지 (정답 1개 + 오답 3개)
- 정답 선택 시: +50포인트 자동 적립
- 오답 선택 시: 정답 표시, 포인트 없음

### 학습 진도 추적

**진도 기록:**
- `correct_count`: 정답 횟수
- `incorrect_count`: 오답 횟수
- `last_practiced`: 마지막 학습 시간
- `mastered`: 마스터 여부 (정답 3회 이상)

**마스터 조건:**
- 정답 3회 이상
- 정답률 75% 이상

### 포인트 적립 규칙

**퀴즈 정답 시:**
- 1회 정답: +50포인트
- 연속 정답 보너스 없음 (중복 적립 방지)

**학습 완료 보너스:**
- 카테고리 10개 완료: +500포인트
- 전체 100개 완료: +5000포인트

### 난이도 시스템

**난이도 1 (쉬움):**
- 일상 생활에서 자주 사용하는 단어
- 짧고 발음하기 쉬운 단어
- 예: dog, cat, red, blue, one, two

**난이도 2 (보통):**
- 조금 더 복잡한 단어
- 긴 단어 또는 추상적 개념
- 예: elephant, purple, watermelon, teacher

**난이도 3-5 (어려움):**
- 현재 미사용 (향후 확장 가능)
- 복잡한 문법, 긴 문장 등

### 학습 추천 알고리즘

**1. 순차 학습:**
- 카테고리별로 순서대로 학습
- 동물 → 색깔 → 음식 → ...

**2. 복습 우선:**
- 오답률이 높은 단어 우선 출제
- 마지막 학습 시간이 오래된 단어 우선

**3. 난이도 조절:**
- 정답률이 높으면 난이도 높은 단어 출제
- 정답률이 낮으면 난이도 낮은 단어 출제

## 학습 시나리오 예시

**시나리오 1: 첫 학습**
1. 카테고리 선택: 동물
2. 플래시카드 모드로 10개 단어 학습
3. 퀴즈 모드로 전환
4. 10문제 중 7문제 정답
5. 포인트 적립: 7 × 50 = 350포인트
6. 학습 진도: 7/10 마스터

**시나리오 2: 복습**
1. 이전에 오답한 3개 단어 복습
2. 퀴즈 모드로 3문제 풀기
3. 3문제 모두 정답
4. 포인트 적립: 3 × 50 = 150포인트
5. 학습 진도: 10/10 마스터 (카테고리 완료)
6. 보너스: +500포인트

**시나리오 3: 전체 완료**
1. 10개 카테고리 모두 완료
2. 100개 단어 마스터
3. 보너스: +5000포인트
4. 배지 획득: "100개 단어 마스터"
